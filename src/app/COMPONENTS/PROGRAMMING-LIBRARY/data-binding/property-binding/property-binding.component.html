PROPERTY BINDING

Property binding is a way to pass data from the component to the view. This is also known as One-Way Data Binding. The reason one would use propety-binding over interpolation is because in property binding, you can pass any kind of data and are not limited to strings only.

Property Binding:


Property bindings are evaluated on every browser event and any changes made to the objects in the event, are applied to the properties. You can use properties binding to control boolean switches. When a property is set to a boolean evaluation, it can be turned on or off by an event that changes its declared value.
