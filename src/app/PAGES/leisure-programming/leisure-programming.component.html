Two-Way Binding

In any angular html template, you can inject text dynamically. Text can be stored within variables in the components.ts file. Through variable replacement, you can inject this text into the variable call of the html. This requires you to do two things:


  1. Declare your variable in the components.ts file: 

                import { Component  } from '@angular/core';

            @Component({
              selector: 'app-root',
              template: `
              <h1>{{title}}</h1>
              <h2>My favorite hero is: { { myHero } } </h2>
                `
                })
            export class AppComponent {
              title = 'Tour of Heroes';
              myHero = 'Windstorm';
              }
  2. Call your variable's name in the template file surrounding it with two angle brackets: 

                  import { Component  } from '@angular/core';
  
              @Component({
                selector: 'app-root',
                template: `
                <h1>{{title}}</h1>
                <h2>My favorite hero is: { { myHero } } </h2>
                  `
                  })
              export class AppComponent {
                title = 'Tour of Heroes';
                myHero = 'Windstorm';
                }
